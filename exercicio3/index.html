Marcadores de Ênfase

        Neste parágrafo utilizaremos diferentes marcadores de ênfase,
        essas marcações permitem criar um texto com maior visibilidade
        para o leitor. Podemos utilizar em  negrito. Além do negrito
        podemos utilizar outro maracdor que permite destaques de texto importantes,
        textos em itálico, texto enfatizado,sobrescrito,texto inserido,
        subscrito,excluído, texto menor, texto marcado.
