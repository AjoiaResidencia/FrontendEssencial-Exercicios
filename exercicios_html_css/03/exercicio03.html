Marcações de ênfase

Neste parágrafo utilizaremos diferentes marcações de ênfase, essas marcações permitem criar um texto com maior visibilidade para o leitor. Podemos utilizar texto em negrito. Além do negrito podemos utilizar outro marcador que permite destaques de textos importantes, textos em itálico, texto enfatizado, sobrescrito, texto inserido, subscrito, Excluído, texto menor, texto marcado.
